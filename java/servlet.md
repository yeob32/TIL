# Java Servlet

## Servlet
- Java 기반의 웹서비스 기술
- 웹페이지를 동적으로 생성하는 서버

### 서블릿 생명주기

### 서블릿 컨테이너
- 서블릿 관리
- 클라이언트의 요청을 받아주고 응답을 할 수 있게, 웹서버와 소켓을 만들어 통신
- ex) 톰캣

### Tomcat
- 웹 애플리케이션 서버 (WAS)
- 웹 프로그래머가 작성한 servlet 관리 (서블릿 컨테이너)
    - 클라이언트의 요청에 대한 servlet 제어 

### JSP (Java Server Page)
- JSP -> servlet 파일 -> 클래스 -> 실행
