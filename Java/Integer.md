# Integer

### byte, short, int, long

```
byte ->  []                      -> 1byte = 8bit  -> -128 ~ 127
short -> [] []                   -> 2byte = 16bit -> -32,768 ~ 32,767
int ->   [] [] [] []             -> 4byte = 32bit -> -2,147,483,648 ~ 2,147,483,647
long ->  [] [] [] [] [] [] [] [] -> 8byte = 64bit
```
